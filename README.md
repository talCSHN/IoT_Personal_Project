### 컬러센서
- TCS3200 아두이노 컬러감지 센서 모듈 GY-31
- RGB 색상을 체크하는데 작게 센싱되는 값이 해당 색상을 의미

<img src="./image/mp0010.jpg" width="500">

- 핀(10개) 단자 설명
    - VCC : 두개 존재. 하나만 사용해도 무방. 3.3V, 5V
    - GND : 두개 GND 연결
    - S0 : D2
    - S1 : D3
    - S2 : D4(!) 문제 발생
    - S3 : D5
    - OUT : D6
    - LED : 연결안함

<img src="./image/mp0011.jpg" width="650">

- 웹캠, AI카메라 장착해서 물체인식, 사진촬영으로 변경가능
